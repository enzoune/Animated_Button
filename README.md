# 🖱️ Interactive Animated Button (CSS Micro-interaction)

> Um componente de botão moderno com animação fluida, desenvolvido exclusivamente com HTML5 e CSS3 para demonstrar domínio em microinterações e design de interface.

## 📖 Sobre o Projeto

Este projeto faz parte do meu portfólio de estudos em **Programação Full-Stack**. O foco aqui foi criar um componente de Call-to-Action (CTA) que utiliza estados de animação para melhorar o feedback visual e o engajamento do usuário.

O projeto demonstra como elementos simples de UI podem ser elevados através de transições suaves e efeitos de preenchimento, essenciais para criar experiências de usuário (UX) memoráveis em sistemas web.

## ✨ Destaques e Diferenciais Técnicos

* **Microinteração com CSS:** Implementação de efeitos visuais no estado de *hover*, demonstrando competência na manipulação de pseudo-elementos e propriedades de transição.
* **Design Encapsulado:** Uso de uma tag `<span>` dentro do link para permitir camadas de animação sobrepostas ou efeitos de texto diferenciados.
* **Organização de Código:** Separação clara entre estrutura (HTML) e estilo (CSS), seguindo as boas práticas de engenharia de software com o uso de arquivos externos.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação do componente.
* **CSS3:** Estilização, gerenciamento de transições e animações de interação.
